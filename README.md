# Stitch.io Web Application

Stitch.io의 웹 애플리케이션 프론트엔드 및 Express 로컬 실행 환경입니다.

## 🚀 시작하기

### 1. 저장소 클론 (Clone)
```bash
git clone https://github.com/YOUR_USERNAME/stitch-io-app.git
cd stitch-io-app
```

### 2. 패키지 설치
```bash
npm install
```

### 3. 로컬 서버 실행
```bash
npm start
```

서버 실행 후 브라우저에서 `http://localhost:3000` 접속 시 웹 애플리케이션을 확인하실 수 있습니다.

## 🛠 기술 스택
- **HTML5 & Vanilla JavaScript**
- **Tailwind CSS** (via CDN)
- **Lucide Icons**
- **Node.js & Express**
