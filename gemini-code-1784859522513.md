# 🫧 서우 여드름 짜기 챌린지 (Pop Pop Seou)

Google Gemini API와 파스텔 네온 네오모피즘 UI 스타일(Stitch.io visual style)을 반영한 캐주얼 미니게임 프로젝트입니다.

---

## 📌 주요 기능

1. **타임어택 모드 (AI 여드름 배치)**
   - 프롬프트를 입력하면 Gemini 2.0 Flash가 부위별 여드름의 크기, 좌표, 제한시간 데이터를 생성합니다.
   - 클릭 및 터치 인터랙션을 통한 파티클 효과와 콤보 시스템을 지원합니다.
2. **반응속도 테스트 모드**
   - 시각 자극에 대한 반응 시간을 밀리초(ms) 단위로 정밀하게 측정합니다.
   - 라운드별 시각화 그래프 및 등급 판정 기능을 지원합니다.

---

## 🛠 실행 및 배포 방법

### 1. Vercel 배포 (추천)
1. GitHub 리포지토리를 Vercel 프로젝트로 연결합니다.
2. Vercel 프로젝트 환경 변수(`Environment Variables`) 설정:
   - `GEMINI_API_KEY`: Google AI Studio에서 발급받은 API Key
3. 배포 완료 후 연동된 도메인으로 사용 가능합니다.

```bash
# Vercel CLI 설치

# 개발 서버 실행
